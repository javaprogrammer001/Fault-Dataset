# Fault-Dataset
This Dataset is created using PSCAD simulations for faults applied at various distances on the AB line of a 220 kV AC system. Measurements are recorded at the relay terminals for fault type and distance analysis.

Fault Types (FT)
1. Line-to-Ground (L-G)
A-G, B-G, C-G
2. Line-to-Line (L-L)
A-B, B-C, C-A
3. Line-to-Line-to-Ground (L-L-G)
A-B-G, B-C-G, C-A-G
4. Three-phase (L-L-L)
A-B-C


Fault Distances (FD)

Faults are applied at:

1. 20 km (10%)
2. 50 km (25%)
3. 100 km (50%)
4. 150 km (75%)
5. 160 km (80%)
